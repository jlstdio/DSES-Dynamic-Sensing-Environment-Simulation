## 📈 주요 통계

### 전력 소비 범위
- **최저**: Idle (434.85 mW)
- **최고**: Transmission (646.55 mW)
- **추론 평균**: ~485 mW (Layer 1-5)

### 레이어별 소요 시간 (평균)
- Layer 0: 38,785 ms (38.8초)
- Layer 1: 45,236 ms (45.2초)  
- Layer 2: 45,567 ms (45.6초)
- Layer 3: 49,054 ms (49.1초)
- Layer 4: 49,305 ms (49.3초)

### 전송 성능
- **Byte당 에너지**: 153.46 mJ/byte (256 bytes 가정)
- **평균 전송 시간**: 57,495 ms (57.5초)
- **전송 전력**: 646.55 mW (모든 모드 중 최고)

## 💡 최적화 제안

1. **센서 선택**: accel+gyro 조합이 가장 에너지 효율적 (439.48 mW)
2. **레이어 효율**: Layer 4가 MFLOP당 에너지가 가장 낮음 (33.75 J/MFLOP)
3. **전송 최적화**: 전송이 가장 많은 전력 소비 → 데이터 압축/배치 전송 고려
4. **대기 시간 최소화**: Idle 상태도 434.85 mW 소비 → 슬립 모드 활용 권장
